Provide ZoodPay as a buy now pay later payment option for WooCommerce

== Description ==

Zoodpay is a buy now pay later solution which allows your customers in the Silk road countries to defer their payments, with no interest at all. Zoodpay is part of the Zoodmall family, one of the biggest Marketplace in MENA and CIS countries. Thanks to Zoodpay, buyers in those regions have a seamless access to 2 payment solutions:

* ZPI : pay for your order in 4 instalments overs 90 days with no interest.
* PAD : pay for your order 14 days after delivery with no interest.

Zoodpay is an amazing payment solution for local merchants as well as Cross border merchants to offer if they want to grow their business and to tap safely into those emerging markets. Zoodpay is currently available for customers in the following markets -


* Kazakhstan (KZ)
* Uzbekistan (UZ)
* Iraq (IQ),
* Jordan(JO)
* Kingdom Of Saudi Arabia(KSA)
* Kuwait(KW)
* Lebanon(LB)


== HOW TO GET STARTED ==
 

In order to use Zoodpay, you will need to have an account with us first, so please contact us at https://www.zoodpay.com/contact in order to sign-up for Zoodpay
Install the plugin on your site. 
Collect the sandbox credentials from ZoodPay team and configure in ZoodPay plugin
Get sign off from ZoodPay team on sandbox to get production credentials.

== Frequently Asked Questions ==


= Which countries does this ZoodPay supported? =

Zoodpay is currently available for customers in the following markets - Kazakhstan (KZ), Uzbekistan (UZ), Iraq (IQ), Jordan(JO), Kingdom Of Saudi Arabia(KSA), Kuwait(KW)


= What is specific requirements? =

WooCommerce 3.0 or newer is required.
PHP 5.6 or higher is required.
A SSL Certificate is required.
This plugin integrates with ZoodPay v0 platform. You need an agreement with Zoodpay specific to the v0 platform to use this plugin.

= What to do if I need any help? =

Email us at integration@zoodpay.com

= Extension installed successfully however services not appearing on payment page? =

Please check the final order amount which needs to be lie between min and max value for that market. If condition matching and still not showing option then log in to your Wordpress Admin panel. Go to ZoodPay Plugin settings and click on Get Configuration button. Refresh the payment page and if still not displaying then please contact to integration@zoodpay.com


= What will happen if ZoodPay plugin doesn't support my WooCommerce? =

Please write your issue/query to integration@zoodpay.com.


== Installation ==

This section describe the steps to install the ZoodPay plugin

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser. To do an automatic install of the ZoodPay plugin, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type “ZoodPay Buy NowPay Later” and click Search Plugins. Once you’ve found our plugin you can view details about it such as the point release, rating, and description. Most importantly, of course, you can install it by simply clicking "Install Now", then "Activate".

= Manual installation =

The manual installation method involves downloading our plugin and uploading it to your web server via your favorite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

= Configuration =

1. Login to your WordPress admin.
1. Navigate to "WooCommerce > Settings".
1. Click the "Checkout" tab.
1. Click the "ZoodPay" sub-tab.
1. Enter the Merchant ID, Secret Key & salt that were provided by ZoodPay for Production use.
1. Save changes.


= Supported Themes =

It support all themes supported by Woo. If themes doesn't support then write to integration@zoodpay.com to customise the plugin according to your theme.
